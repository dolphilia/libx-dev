# はじめに

このドキュメントはテンプレートプロジェクトです。新しいドキュメントサイトを作成する際の出発点として使用してください。

## このテンプレートについて

このテンプレートには以下の機能が含まれています：

- **多言語対応**: 英語と日本語でのコンテンツ作成
- **バージョン管理**: 複数バージョンの同時管理
- **自動サイドバー**: コンテンツ構造に基づく自動ナビゲーション
- **共通UIコンポーネント**: 統一されたデザインシステム
- **レスポンシブデザイン**: すべてのデバイスに対応

## プロジェクト構造

```
src/content/docs/
├── en/                    # 英語ドキュメント
│   ├── v1/               # バージョン1
│   └── v2/               # バージョン2
│       ├── 01-guide/     # ガイド
│       ├── 02-components/ # コンポーネント
│       ├── 03-advanced/  # 上級者向け
│       └── 04-reference/ # リファレンス
└── ja/                   # 日本語ドキュメント
    ├── v1/
    └── v2/
```

## カスタマイズ方法

1. **プロジェクト設定**: `src/config/project.config.json`を編集
2. **表示名の変更**: 各言語の`displayName`を更新  
3. **コンテンツ追加**: `src/content/docs/`以下にMDXファイルを作成
4. **カテゴリ設定**: サイドバーのカテゴリを定義

## 開発の開始

新しいドキュメントサイトの開発を開始する前に：

- [ ] プロジェクト名を決定
- [ ] 対象バージョンを決定
- [ ] コンテンツ構成を計画
- [ ] 既存のテンプレートコンテンツを削除・更新

## 次のステップ

このテンプレートを使って新しいプロジェクトを始めましょう：

1. 既存のサンプルコンテンツを削除
2. プロジェクト固有のドキュメントを作成
3. 設定ファイルをプロジェクトに合わせて調整
4. テストとビルドを実行

テンプレートとしての役割を理解し、実際のプロジェクトに合わせてカスタマイズしてください。
