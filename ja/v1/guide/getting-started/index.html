入門

プラットフォームの使い方を学ぶ

はじめに

ドキュメントへようこそ！このガイドでは、プラットフォームの使い方について説明します。

前提条件

  • Node.js 18以上
  • npm、yarn、またはpnpm
  • JavaScriptの基本的な知識

インストール

npmを使用してパッケージをインストールできます：

npm install @our-package/core

または、yarnを使用：

yarn add @our-package/core

または、pnpmを使用：

pnpm add @our-package/core

基本的な使い方

パッケージの使用例：

import { createApp } from '@our-package/core';

const app = createApp({
  // 設定オプション
  debug: true,
  mode: 'development'
});

app.start();
